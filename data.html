Consumer Behavior Prediction

Using Python and Random Forest models to analyze purchase paths and optimize conversion rates for e-commerce platforms.

Python Scikit-learn SQL
Objective

To identify the key behavioral triggers that lead a casual browser to become a repeat customer, using a dataset of 50,000+ user sessions.

[ Data Visualization: Conversion Funnel Chart will be here ]

Tools: Matplotlib / Seaborn

The Methodology

I applied an **XGBoost regressor** to rank feature importance. Interestingly, the time spent on "Shipping Policy" pages was a stronger predictor of conversion than "Discount" page views—a classic "Data Storytelling" insight that shifted our marketing focus.

Data Cleaning

Handling missing values and outliers in session duration using RobustScaler.

Model Accuracy

Achieved an F1-score of 0.88 through hyperparameter tuning with GridSearchCV.
